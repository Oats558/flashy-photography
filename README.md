# Flashy Photography (Student Demo Site)

This is a simple static website created for IT299 final product submission.

## Pages
- Home: `index.html`
- Portfolio: `portfolio.html`
- Booking: `booking.html` (demo form saves to localStorage)
- Contact: `contact.html`

## How to run locally
Open `index.html` in a browser.

## How to deploy (easy option)
1. Create a free GitHub account (if needed)
2. Create a new repo
3. Upload everything inside the `code` folder
4. Enable GitHub Pages (Settings -> Pages)
5. Your live URL will appear in GitHub Pages settings.

(You can also use Netlify/Vercel if preferred.)
